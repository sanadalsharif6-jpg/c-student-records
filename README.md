# C Student Records Pro

A stronger C student records system with binary persistence and reporting.

## Features

- Add students
- Update students
- Delete students
- Search by ID
- Weighted average calculation
- Letter grades
- Class report
- Export text report
- Binary file persistence

## Compile

```bash
gcc main.c -o student_records
```

## Run on Windows PowerShell

```bash
.\student_records.exe
```

## Run on macOS/Linux

```bash
./student_records
```

## Publish to GitHub

```bash
git init -b main
git add .
git commit -m "Initial commit"
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```
